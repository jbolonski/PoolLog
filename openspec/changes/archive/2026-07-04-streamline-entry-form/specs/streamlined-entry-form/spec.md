## ADDED Requirements

### Requirement: Streamlined entry form reveals input fields immediately
The system SHALL present the pool reading input fields in the primary entry area when the user opens the log form so the user can begin entering values without scrolling.

#### Scenario: Entry form opens for a new reading
- **WHEN** the user taps the log action
- **THEN** the form fields appear in the main visible area of the screen without requiring additional scrolling

#### Scenario: User completes a reading entry
- **WHEN** the user enters values into the visible input fields
- **THEN** the form remains focused on the entry experience and the user can submit the reading without navigating away from the main form area
