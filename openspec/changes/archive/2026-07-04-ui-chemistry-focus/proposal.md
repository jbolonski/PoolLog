## Why

The current interface mixes chemistry logging with general maintenance content, which dilutes the app's focus. Simplifying the screen so the experience centers on the most important water readings will make the app feel clearer and more useful for daily pool care.

## What Changes

- Remove the Today section from the home experience.
- Remove the Routine section from the home experience.
- Remove the Tasks entry from the bottom navigation bar.
- Simplify the header title so it feels more minimal.
- Keep the chemistry readings and logging experience as the primary focus.

## Capabilities

### New Capabilities
- chemistry-focused-ui: present a streamlined interface centered on pool chemistry readings.

### Modified Capabilities
- None.

## Impact

- Main application layout and navigation.
- Visual hierarchy of the home screen.
- User attention toward chemistry logging rather than general maintenance tasks.
