## Requirements
### Requirement: Present a chemistry-first layout
The system SHALL present the pool chemistry experience as the primary focus of the home screen.

#### Scenario: Home screen removes secondary sections
- **WHEN** a user opens the app
- **THEN** they SHALL see chemistry readings and logging controls without a Today or Routine section.

### Requirement: Simplify navigation and header
The system SHALL provide a minimal header and navigation experience that does not include a Tasks entry.

#### Scenario: Bottom navigation omits Tasks
- **WHEN** a user views the main navigation
- **THEN** the navigation SHALL not include a Tasks entry.
